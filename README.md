# 2\. 项目标语 (Tagline)

# &nbsp;	当古老咒语遇见赛博朋克：一个能“听”到律动的沉浸式诵经工具。

# &nbsp;	Ancient Mantra meets Cyberpunk Aesthetics: An immersive, rhythmic chanting experience.

# 3\. 项目简介 (About / Description)

# &nbsp;	CyberZen Reader 是一款专为《大佛顶首楞严神咒》设计的沉浸式律动阅读器。它打破了传统经文阅读的枯燥感，通过 Web Audio API 实时合成类似 Sonic Pi 的深沉电子律动，配合独特的 4-6-8 句式循环排版，帮助诵读者在极具未来感的赛博夜景中进入深度冥想状态。

# 如果你厌倦了传统的木鱼声，想要一种更具现代感、更能带动呼吸节奏的修行方式，这个工具就是为你准备的。

# 4\. 核心亮点 (Key Features) - 这是吸引人的关键

# &nbsp;	🧘 4-6-8 节律断句法：独创算法将经文自动切割为 4 句、6 句、8 句的循环段落，完美适配呼吸起伏，让长咒诵读不再气短。

# &nbsp;	🎹 Sonic Pi 级音频引擎：内置 6 种实时合成音色，包括：

# &nbsp;	深度脉冲 (Sub-Bass)：震动胸腔的超低音。

# &nbsp;	雷击顿 (Reggaeton)：充满活力的 3-3-2 律动。

# &nbsp;	Sam Aaron 经典复刻：包含著名的 Square Skit 与 Tilburg 2 工业风节拍。

# &nbsp;	🌃 赛博朋克视觉美学：全屏无限位移的城市夜景背景，配合 5x5 动态网格与毛玻璃（Glassmorphism）卡片排版。

# &nbsp;	🚀 极简单文件架构：纯 HTML/JS 实现，无任何外部库依赖，零延迟加载，打开即用。

# &nbsp;	💾 智能交互：内置一键跳转“起句”逻辑，以及基于 localStorage 的个性化名片缓存系统。
